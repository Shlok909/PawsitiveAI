# **App Name**: Pawsight AI

## Core Features:

- Onboarding Flow: Streamlined onboarding with Firebase Auth, including email and Google sign-in, dog profile creation, and permissions requests.
- AI-Powered Emotion Analysis: Analyze dog's video or image to detect their emotion and confidence level. The tool is a veterinary behaviorist using a JSON schema.
- Insights Report Generation: Generate detailed insights reports including body language breakdown, health vitals, and action items.
- AI Chat Assistant: Implement an AI chat assistant that provides context-aware responses and personalized advice based on the analysis reports.
- Mood Tracking and History: Maintain a history of past analyses with mood trends and filtering options.
- Fullscreen Camera Analysis Lab: Develop an analysis lab with a fullscreen camera preview with guidelines to record videos and upload images of the dog for analysis.
- Responsive Home Dashboard: A dashboard that shows the dog's current mood, recent reports, and an action button to start a new analysis.

## Style Guidelines:

- Primary color: Trustworthy blue (#4A90E2) to establish credibility.
- Secondary color: Health green (#50C878) for positive reinforcement in reports.
- Background color: Desaturated blue (#E8F0FE), providing a calm and trustworthy backdrop that complements the primary color without overwhelming the user interface.
- Body and headline font: 'PT Sans', a humanist sans-serif for both headlines and body text.
- Modern icons to represent different dog emotions and health indicators.
- Neumorphism cards with soft shadows for a modern, user-friendly feel.
- Smooth page transitions and subtle loading animations to improve UX.
